This rule checks for instances where CACHE.DBR can be replaced by CACHE.READ.

Noncompliant Code Example

CALL CACHE.DBR('ACCOUNT',AC.CURRENCY,ACCOUNT,ACCOUNT.CCY)
EB.DataAccess.CacheDbr('ACCOUNT',AC.CURRENCY,ACCOUNT,ACCOUNT.CCY)

Compliant Solution

CACHE.READ should be used instead of CACHE.DBR.

CALL CACHE.READ('F.ACCOUNT', AccId, Record, Er)
EB.DataAccess.CacheRead('F.ACCOUNT', AccId, Record, Er)
