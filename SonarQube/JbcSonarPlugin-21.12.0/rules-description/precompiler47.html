This rule provides the name of API that can be used to read ECB record for ACCOUNT application.

Noncompliant Code Example

RE.ConBalanceUpdates.EbContractBalances.Read(RecId, Error)
CALL F.READ('F.EB.CONTRACT.BALANCES',RecId,Rec,'',Error)

Compliant Solution

Use respective API to read contract balances record for ACCOUNT.

AC.API.EbReadContractBalances(AcctNo, EbContractBalRec, Yerr, RecordLock)
