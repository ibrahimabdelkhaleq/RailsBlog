This rule provides the name of API that can be used to read record from CURRENCY table.

Noncompliant Code Example

ST.CurrencyConfig.Currency.Read(RecId, Error)
CALL F.READ('FBNK.CURRENCY',RecId,Rec,'',Error)

Compliant Solution

Use API to get the currency record instead of direct read.

ST.CurrencyConfig.GetCurrencyRecord(CcyCompMne, RatesId, RatesRecord, ErrMsg)
