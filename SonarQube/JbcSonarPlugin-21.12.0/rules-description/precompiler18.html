This rule checks for instances where DBR can be replaced by F.READ.

Noncompliant Code Example

CALL DBR('ACCOUNT':FM:AC.MASTER.ACCOUNT,ACC.NO,MAST.ACC.NO)
EB.DataAccess.Dbr('ACCOUNT':FM:AC.MASTER.ACCOUNT,ACC.NO,MAST.ACC.NO)

Compliant Solution

Consider reading the full record using F.READ instead of DBR.

CALL F.READ(FN.ACCOUNT, AccId, Rec, F.ACCOUNT, Er)
EB.DataAccess.FRead(FN.ACCOUNT, AccId, Rec, F.ACCOUNT, Er)
