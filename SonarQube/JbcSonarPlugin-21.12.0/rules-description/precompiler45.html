This rule provides the name of API that can be used to read the history record of CURRENCY table.

Noncompliant Code Example

ST.CurrencyConfig.Currency.ReadHis(RecId, Error)
CALL F.READ('FBNK.CURRENCY$HIS',RecId,Rec,'',Error)

Compliant Solution

Use API to get the history file instead of direct read on history file.

ST.CurrencyConfig.GetCcyHistory(HisDate, Yccy, CcyDateRec, ReturnCode)
