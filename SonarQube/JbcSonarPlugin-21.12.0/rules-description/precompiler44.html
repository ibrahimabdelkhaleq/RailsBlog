This rule identifies the list of commands which are not supported in TAFJ.

Noncompliant Code Example

EXECUTE 'SELECT F.ACCOUNT$HIS'
EXECUTE 'LIST-ITEM FBNK.CUSTOMER'

Compliant Solution

Avoid using the following list of commands as they are not supported in TAFJ. ESEARCH, SEARCH, LIST, LIST-ITEM, REFORMAT, SREFORMAT, SDUMP, TLOAD, SORT, SORTITEM, SORTLABEL, STAT, SUM, TDUMP, TLOAD.
