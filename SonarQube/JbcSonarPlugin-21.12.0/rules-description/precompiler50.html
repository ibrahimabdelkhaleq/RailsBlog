This rule provides an info to replace all the EB.READLIST calls with DAS.

Noncompliant Code Example

CALL EB.READLIST(SEL.CMD,LIST.KEY1,'',NO.SEL,'')
EB.DataAccess.Readlist(SelectStatement, KeyList, ListName, Selected, SystemReturnCode)

Compliant Solution

Consider using DAS instead of EB.READLIST.

CALL DAS(TableName, TheList, TheArgs, TableSuffix)
EB.DataAccess.Das(TableName, TheList, TheArgs, TableSuffix)
