This rule makes it mandatory to open all the necessary files in the .LOAD routine of a multi-threaded job instead of record routine.

Noncompliant Code Example

SUBROUTINE TEST
EB.DataAccess.Opf(FN.OFS.SOURCE.VAL, F.OFS.SOURCE.VAL)
CALL OPF(FN.OFS.SOURCE.VAL, F.OFS.SOURCE.VAL)

Compliant Solution

All necessary files should only be opened in the '.LOAD' of a multi-threaded job.

SUBROUTINE TEST.LOAD
EB.DataAccess.Opf(FN.OFS.SOURCE.VAL, F.OFS.SOURCE.VAL)
CALL OPF(FN.OFS.SOURCE.VAL, F.OFS.SOURCE.VAL)
