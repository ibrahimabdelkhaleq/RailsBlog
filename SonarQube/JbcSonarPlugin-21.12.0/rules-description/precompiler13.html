This rule restricts the usage of IDESC as it is not supported in TAFJ.

Noncompliant Code Example

CALL IDESC(FILE, ID, RECORD, IDESC, RESULT)
EB.API.Idesc(File, Id, Record, Idesc, Result)

Compliant Solution

IDESC is not supported in TAFJ. Alternatively, handle the required functionality, by attaching a conversion routine to the field, in enquiry.
