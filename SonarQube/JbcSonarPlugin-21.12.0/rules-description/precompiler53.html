This rule provides the name of available SMS API's that can be used to get the details of SMS profile and role information.

Noncompliant Code Example

EB.Security.UserRoles.Read(RecId, Error)
EB.DataAccess.Opf('F.EB.USER.ROLES','')

Compliant Solution

Use respective APIs instead of direct read on the table.

EB.Sms.getRole(userId, companyId, roleId)
EB.Sms.getSmsProfile(roleId, smsProfile)
