This rule makes it mandatory to use session number instead of terminal number.

Noncompliant Code Example

EB.SystemTables.setTno('123')
R.BATCH<18> = TNO

Compliant Solution

C$SESSION.NO should be used in place of TNO.

R.BATCH<18> = C$T24.SESSION.NO
R.BATCH<18> = EB.SystemTables.getCTTwoFouSessionNo()
