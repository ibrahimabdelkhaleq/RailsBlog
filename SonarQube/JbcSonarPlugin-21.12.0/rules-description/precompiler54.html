This rule restricts the usage of OfsGlobusManager.

Noncompliant Code Example

EB.Interface.OfsGlobusManager(KOfsSource, OfsMessage)
CALL OFS.GLOBUS.MANAGER(KOfsSource, OfsMessage)

Compliant Solution

OfsCallBulkManager should be used instead of OfsGlobusManager.

EB.Interface.OfsCallBulkManager(Options, Therequest, Theresponse, Txncommitted)
CALL OFS.CALL.BULK.MANAGER(Options, Therequest, Theresponse, Txncommitted)
