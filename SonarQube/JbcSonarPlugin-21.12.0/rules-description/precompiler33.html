This rule is used to make sure that TEXT is set with proper error format before making a call to STORE.OVERRIDE.

Noncompliant Code Example

TEXT = 'AA.CUSTOMER.MISMATCH'
CURR.NO = 0 
CALL STORE.OVERRIDE (CURR.NO)

Compliant Solution

Set TEXT with pattern (Product-OverrideText)

TEXT = 'AA-CUSTOMER.MISMATCH'
CURR.NO = 0 
CALL STORE.OVERRIDE (CURR.NO)
