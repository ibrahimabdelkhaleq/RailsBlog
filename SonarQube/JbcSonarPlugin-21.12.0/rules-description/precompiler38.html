This rule states that there should be no codes in the same line after THEN/ELSE statements.

Noncompliant Code Example

IF TEST.VAR THEN X = 6 ELSE X = 7
IF TEST.VAR ELSE preserveOption = ''

Compliant Solution

Code should be written in separate lines after THEN and ELSE statements.

IF TEST.VAR THEN 
X = 6
END ELSE 
X = 7
END
