This rule makes it mandatory to use the Account service APIs available to perform operations on Account records instead of direct access to Account table.

Noncompliant Code Example

bal(AC.AccountOpening.Account.OpenActualBal) = 1000
openactbal = bal(AC.AccountOpening.Account.OpenActualBal)

Compliant Solution

Use the respective APIs to get the balance details for ACCOUNT.

CALL Accountmulti-threaded job.setOpenActualBalance(ACCOUNT.NO, YBALANCE, response)
CALL Accountmulti-threaded job.getOpenActualBalance(ACCOUNT.NO, YBALANCE, response)
