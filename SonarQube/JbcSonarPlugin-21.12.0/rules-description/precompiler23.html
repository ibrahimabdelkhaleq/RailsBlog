This rule provides an info to replace TXT with overrides as TXT is not supported.

Noncompliant Code Example

CALL TXT('Hello')

Compliant Solution

Consider the usage of STORE.OVERRIDE instead of TXT. Assign the overrideId to the common variable TEXT before calling STORE.OVERRIDE. 

TEXT = 'Hello'
CURR.NO = 0 
CALL STORE.OVERRIDE (CURR.NO)

Exceptions

TXT holds good only for command line processing.
