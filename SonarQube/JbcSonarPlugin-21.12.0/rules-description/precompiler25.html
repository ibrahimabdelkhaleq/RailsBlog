This rule restricts DCOUNT on FOR loop as the performance gets affected when DCOUNT is used on each iteration.

Noncompliant Code Example

FOR I = 1 TO DCOUNT(TestVar,@FM)
NEXT I

Compliant Solution

Use a separate variable to get the value of DCOUNT and then use the same variable in FOR loop.

CNT = DCOUNT(TestVar,@FM)
FOR I = 1 TO CNT
NEXT I
