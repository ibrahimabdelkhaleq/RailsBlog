This rule is used to indicate that BATCH.BUILD.LIST call must be made in SELECT routine of a multi-threaded job atleast once.

Noncompliant Code Example

SUBROUTINE TEST.SELECT
RETURN

Compliant Solution

Call to BATCH.BUILD.LIST must be made only in the '.SELECT' routine of a multi-threaded job.

SUBROUTINE TEST.SELECT
EB.multi-threaded job.BatchBuildList(ListParameters, IdList)
CALL BATCH.BUILD.LIST(ListParameters, IdList)
