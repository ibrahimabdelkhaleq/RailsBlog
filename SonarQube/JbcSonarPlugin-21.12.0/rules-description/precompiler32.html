This rule is used to make sure that ETEXT is set with proper error format before making a call to STORE.END.ERROR.

Noncompliant Code Example

ETEXT ='EB.RTN.DUP'
CALL STORE.END.ERROR

Compliant Solution

Set ETEXT with pattern (Product-ErrorText)

ETEXT = 'EB-TEST.CORRECT.FORMAT'
CALL STORE.END.ERROR
