This rule makes it mandatory to use the Customer service APIs available to perform operations on Customer records instead of direct access to Customer table.

Noncompliant Code Example

CALL OPF('F.CUSTOMER','')
ST.Customer.Customer.Read(RecId, Error)

Compliant Solution

Use Customer service APIs to get the customer details.

ST.Customermulti-threaded job.getRecord(CusId,CustomerRec) ;*Getting the customer Record
CALL Customermulti-threaded job.getLanguage(customerKey, customerLanguage)
