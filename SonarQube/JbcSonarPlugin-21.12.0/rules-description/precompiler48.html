This rule provides the name of API that can be used to write ECB record for ACCOUNT application.

Noncompliant Code Example

RE.ConBalanceUpdates.EbContractBalances.Write(RecId, Error)
CALL F.WRITE('F.EB.CONTRACT.BALANCES',RecId,Rec,'',Error)

Compliant Solution

Use respective API to write contract balances record for ACCOUNT.

AC.API.EbWriteContractBalances(AcctNo, EbContractBalRec)
