This rule restricts the usage of Input/Output operations in .FILTER routine of a multi-threaded job.

Noncompliant Code Example

SUBROUTINE TEST.FILTER
ST.Customer.Customer.CacheRead(RecId, Error)

Compliant Solution

All I/O operations should be done only in record routine of a multi threaded job.

SUBROUTINE TEST
ST.Customer.Customer.CacheRead(RecId, Error)
